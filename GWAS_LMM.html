Reference

Single-locus model and their respective software and packages for GWAS using LMM

GRAMMAR

GRAMMAR, an alternate method to pedigree-founded QTL association mapping, which is a quick and easy method. It can handle millions of markers and significantly faster than the evaluated genotype approach for association analysis.

Aulchenko et al., 2007a

EMMA

EMMA is a fixed model edition of LMM using for controlling population structure and genetic relatedness for GWAS.

http://mouse.cs.ucla.edu/emma/ class="s8">

Kang et al., 2008

CMLM/P3D 

CMLM (compressed MLM) diminished the actual sample size into groups using clustering and used another method, P3D (population parameters previously determined), that removes the re-calculation of variance components. The combined application of these two methods prominently abridged computing time and retained/enhanced statistical power.

Zhang et al., 2010

EMMAX

EMMAX is a variance component approach founded on the LMM method, which decreased the computational time for dissecting big GWAS data sets and used for fixing sample structure in GWASs.

http://genetics.cs.ucla.edu/emmax/

Kang et al., 2008

FaST-LMM

FaST-LMM, an LMM based method, used the subset of markers to manage the polygenic effect resulting in attained accelerate speed and need less memory for GWAS.

https://github.com/fastlmm/FaST-LMM/

Lippert et al., 2011

FaST-LMM-Select

FaST-LMM-Select is a simple method that showed that wisely choosing a reduced number of SNPs consistently enhances power, expands standardization, and decreases computational time.

http://mscompbio.codeplex.com/

Listgarten et al., 2012

GRAMMAR-Gamma

GRAMMAR-Gamma is an exceptionally fast variance component-based method that can be used for the massive human cohort in GWAS. It is established based on the analytical approximation within the context of the score test method.

-

Kang et al., 2008

WarpedLMM

WarpedLMM is a simplification of the ordinary LMM that estimates an ideal transformation from the monitored data for genetic study. Subsequently, the power and accuracy of this method increased in GWAS.

-  

Fusi et al., 2014

ECMLM

ECMLM, enriched CMLM, used various algorithms and then select the most effective mixture between relationship algorithm and grouping algorithm resulting power increased and can utilize on very divers of complex traits in GWAS.

-  

Li et al., 2014

SUPER

SUPER method intensely decreases the number of genetic markers utilized to define individual relationships resulting in computationally fast and outstandingly gains statistical power despite utilizing the whole set of SNPs. 

-  

Wang et al., 2014

RMLM 

RMLM, random-SNP-effect MLM, considers the SNP-effect as random, and it permits to use of Bonferroni correction to estimate the p-value for significance tests.

- 

Wang et al., 2016

GMMAT

GMMAT is an R package for carrying out association tests using GLMMs in GWAS and sequencing association studies.

-

Chen et al., 2016

LMM-Score

LMM-Score, a new method proposed to identify the genetic loci of complex traits. The simulation study showed that the power of this method increased and need less computing time in comparison to the traditional LMM method.

-

Chang et al., 2019
